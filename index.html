What is NoSQL?

  • Not Only SQL
  • group of data storage solutions that are not tabular
Originally NoSQL was used to refer to databases that did not use SQL or as 'not sql'. Over time, the phrase has evolved to mean Not Only SQL. As a broad definition, NoSQL describes a group of storage solutions that are not tabular like traditional relational databases.

Types of NoSQL solutions

  • Key-value
  • Document
  • Graph
  • Column Family

Key-value Stores

    Examples
  • memcached
  • DynamoDB
  • Redis

Advantages/Uses

  • Quick Lookups

Document Databases

    Examples
  • CouchDB
  • RavenDB
  • DocumentDB
  • MongoDB
    Use Cases

Column Family Databases

Examples

  • HBase
  • Amazon Simple DB

Use Cases
